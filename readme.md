# Flask--大红豆商城 项目介绍 :
- 使用**flask**框架
- 使用**mysql**数据库
- 使用**github**管理项目
- 使用**redis**数据库缓存用户session数据
- 项目分为前台 , 后台 ,采用前后端不分离的模式
- ....
## 主要实现功能
1. 用户的 登录/注册
2. 展示首页,商品详情页
3. ...购物车,收藏..
